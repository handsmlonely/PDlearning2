### 产品套路
  做产品是否需要标准流程，这是一个值得讨论的问题。标准流程通常是通过总结前人的经验，持续精进完成的流水线方法。作为非常厉害的产品大师，可能会对套路和流程嗤之以鼻，但那是极少有的产品大师才有的资格。作为多数产品经理从业者，能把流程按部就班的执行，至少保证了自己不会出错，并完成了产品经理的基本要求。
