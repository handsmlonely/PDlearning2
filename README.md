### 产品套路
  做产品是否需要标准流程，这是一个值得讨论的问题。标准流程通常是通过总结前人的经验，持续精进完成的流水线方法。顶尖的产品大师可能会对套路和流程嗤之以鼻，但那是极少人才有的资格。作为多数产品经理从业者，能把流程按部就班的执行，至少保证了自己不会出错，达到最基本的及格线，所以在开始产品工作之前，先从套路学起，会是一个入门的捷径。
